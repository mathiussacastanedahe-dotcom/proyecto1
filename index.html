Temas y Tecnologías Principales

HTML5

Aprende la estructura básica de la web y las últimas etiquetas semánticas.

CSS3

Domina estilos modernos, incluyendo Flexbox, Grid y animaciones.

JavaScript

Crea páginas interactivas, maneja el DOM y desarrolla aplicaciones web dinámicas.

Bootstrap

Framework CSS para desarrollar sitios responsivos de forma rápida y eficiente.

TypeScript

Un superset de JavaScript que añade tipado estático para código más robusto y escalable.

React

Librería de JavaScript para construir interfaces de usuario basadas en componentes.

Sass/SCSS

Preprocesador CSS que permite usar variables, funciones y anidamiento para optimizar estilos.

Git

Sistema de control de versiones esencial para el trabajo colaborativo y el seguimiento de cambios.
